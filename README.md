# Astrotune

**Astrotune** is an open-source and highly customizable offline music player.

## Features
- Play your music library offline
- Customizable interface and themes
- Cross-platform desktop support (Windows, macOS, Linux)
- Built with Tauri for performance and security
- No tracking, no ads

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run in development mode: `npm run tauri dev`
4. Build for release: `npm run tauri build`

## Customization
Astrotune is designed to be easily extended and themed. Modify the UI, add plugins, or change the look to fit your style.
