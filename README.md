# Astrotune 🎵

A modern, customizable music player built with Tauri, TypeScript, and Alpine.js.

## Features

- 📁 **Browse and play music** from any folder on your computer
- 🎨 **Custom themes** - Import your own CSS files to personalize the player
- 💾 **Persistent settings** - Your music folder and theme choices are saved
- 🎵 **Multiple format support** - Plays MP3, WAV, FLAC, and OGG files

## Custom Themes

Astrotune supports fully customizable themes! 

### Quick Start with Themes
1. Click the "🎨 Import Custom Theme" button in the app
2. Select a CSS file
3. Enjoy your personalized music player!
